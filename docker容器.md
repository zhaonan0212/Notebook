## docker容器

#### 1.什么是虚拟化技术

```
所谓的虚拟化技术是一种资源管理技术,他可以将我们计算机中的实体资源进行虚拟化.
虚拟化技术种类很多:软件虚拟化,硬件虚拟化,内存虚拟化,网络虚拟化,桌面虚拟化,服务虚拟化...
```

目的:就是为了提供我们计算机的使用率

#### 2.三种虚拟化框架

```
hardware layer:硬件
host OS:操作系统
virtual macheina manager:虚拟层
```

